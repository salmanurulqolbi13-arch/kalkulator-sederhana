# MINTCALC
## PROJECT AKHIR SEMESTER 

### Deskripsi 
MintCalc adalah kalkulator sederhana dengan History, Theme Toggle & Keyboard Support. MintCalc merupakan kalkulator berbasis web yang dibuat menggunakan HTML, CSS, dan JavaScript. Project ini tidak hanya menyediakan fungsi perhitungan dasar, tetapi juga dirancang dengan tampilan modern neumorphism, mendukung mode terang dan gelap, serta dilengkapi panel histori interaktif untuk meningkatkan pengalaman pengguna.

### Latar Belakang

Kalkulator merupakan salah satu aplikasi dasar yang sangat penting dalam dunia komputasi. MintCalc bertujuan untuk:

* Melatih pemahaman **DOM JavaScript**
* Menerapkan **event handling**
* Mengelola **tema menggunakan CSS Variables**
* Menggabungkan logika program dengan **desain antarmuka modern**

Project ini juga menekankan keseimbangan antara fungsi, tampilan, dan kenyamanan pengguna.

### Fitur-fitur yang ada
1. Kalkulator

    * Operasi dasar: + - × ÷
    * Desimal & multi-digit

2. Tombol:
    
    * DEL → hapus satu karakter
    * C → reset
    * % → mengubah angka menjadi desimal
    * RESULT → menampilkan hasil

3. History

4. Tema & UI
    
    * Light mode & Dark mode
    * Desain neumorphism modern
    * Display dinamis (awal di tengah, aktif di bawah)
    * Scroll otomatis untuk angka panjang
    * Responsive (mobile & desktop) 

5. Keyboard Support
6. Desain & Tema
7. Struktur Project

### Tujuan

Tujuan dibuatnya MintCalc ini adalah:

* Mengembangkan kalkulator web yang **fungsional dan interaktif**
* Menerapkan **dark/light mode** secara dinamis
* Membuat sistem **history perhitungan** yang dapat digunakan kembali
* Mengasah kemampuan dalam **HTML, CSS, dan JavaScript**

### Manfaat

* Membantu pengguna melakukan perhitungan secara cepat
* Memberikan pengalaman visual yang nyaman melalui tema ganda
* Menjadi media pembelajaran untuk memahami pengembangan web dasar

### Ruang Lingkup

MintCalc mencakup:

* Operasi matematika dasar
* Tampilan neumorphism responsive
* Input melalui tombol dan keyboard
* Manajemen histori perhitungan
* Pengaturan tema gelap dan terang
